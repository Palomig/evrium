1. Найдите файл: geometry-guide-v2.0.zip

3. Выберите "Извлечь всё..." или "Extract here"
4. Укажите папку для распаковки

🚀 ШАГ 3: ЗАПУСК
─────────────────

1. Откройте папку с распакованными файлами
2. Найдите файл: index.html
3. Дважды щёлкните по нему
   ИЛИ
4. Щёлкните правой кнопкой → "Открыть с помощью" → выберите браузер

Рекомендуется: Google Chrome или любой современный браузер


✨ ШАГ 4: ПЕРВЫЙ ЗАПУСК
────────────────────────

1. Откроется страница с заголовком "📐 Геометрия 7-9 класс"
2. Слева - меню навигации
3. Начните с раздела "Введение"
4. Попробуйте интерактивные задания!
