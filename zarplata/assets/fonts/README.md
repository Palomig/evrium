# Шрифты

## Klavika Basic Bold Cyrillic

Для корректной работы системы необходимо добавить файлы шрифта **Klavika Basic Bold Cyrillic** в эту папку.

### Необходимые файлы:

1. `klavika-bold.woff2` - современный формат (приоритетный)
2. `klavika-bold.woff` - резервный формат для старых браузеров

### Где получить файлы:

Klavika Basic Bold Cyrillic - это коммерческий шрифт. Файлы шрифта можно получить:
- Приобрести лицензию на официальном сайте
- Использовать существующую лицензию, если она у вас есть
- Конвертировать из других форматов (.ttf, .otf) в .woff2 и .woff

### Как добавить файлы:

1. Скопируйте файлы `klavika-bold.woff2` и `klavika-bold.woff` в эту папку
2. Перезагрузите страницу браузера
3. Шрифт должен автоматически применится ко всем страницам системы

### Проверка работы:

После добавления файлов откройте любую страницу системы и проверьте в инструментах разработчика (F12), что шрифт Klavika загружается корректно.

**Путь к файлам:** `/zarplata/assets/fonts/`

---

**Примечание:** Система уже настроена для использования Klavika во всех компонентах интерфейса. После добавления файлов шрифт будет применен автоматически.
